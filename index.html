ℹ CSV format & help

Contacts load from contacts.csv next to this page (sync with npm run outreach:sync). One row per organization.

Required: orgName — organization or event name.

Recommended: location, type (bike, ski, gear, sports, or other), email, phone, facebook, linkedinPeople, website, eventSpecificLine, notes.

Optional: sourceId — unique ID per row (auto-generated if omitted).

Track outreach to nonprofit gear swap organizers — phone calls, emails, and Facebook messages — from a simple contact list. Pick a region from the dropdown above to get started.

Find contacts (built-in)

The swap-finder CLI in this repo uses your ANTHROPIC_API_KEY from .env with Claude web search to discover swap events and enrich contact info. From the project root:

npm install && npm run build npm run discover # Pass 1 — find events npm run enrich # Pass 2 — find emails, phones, social npm run outreach:sync # copy contacts.csv into this app

Then reload this page (or run npm run outreach:serve for local preview). Check progress anytime with npm run start -- status.
